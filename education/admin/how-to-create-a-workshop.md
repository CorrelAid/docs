# How to create a workshop

<figure><img src="../../.gitbook/assets/Event_anlegen.png" alt=""><figcaption></figcaption></figure>

### 1. Planning

* [ ] Create cards in PM-tool (milestone for workshop itself as well as to do's for creating and reviewing materials)
* [ ] Clarification of&#x20;
  * Date&#x20;
  * Speaker&#x20;
  * Title&#x20;
  * Teaser&#x20;
  * Short description&#x20;
* [ ] Think about how to collect feedback for your event! Is there a [kobo form](https://kobo.correlaid.org/) that you can resuse?

### 2. Publication

* [ ] Create Zoom link
* [ ] Create Pretix event
* [ ] Add event to [CorrelAid website event calendar](https://www.correlaid.org/veranstaltungen/?viewType=list)
* [ ] Create an entry in public calendars:
  * [D64](https://kontakt.d-64.org/ticker/veranstaltungsliste-d64-ticker/)
  * [CDL Civic Data Cafe Calendar](https://app.gitbook.com/u/yypxJZaOYGWiBTx0gsjoJxOUEC12)

### 3. Advertising

* [ ] Use the workflow in #pr-post-this-for-me to initiate the promotion of the event (e.g. newsletter, social media, etc.)
* [ ] Post information in Slack&#x20;

### 4. During the Event &#x20;

* [ ] Document participants (screenshot and asap after the workshop in Pretix)
* [ ] Share feedback link with participants

### 5. Follow-up&#x20;

* [ ] Send materials and feedback link to participants (template in Education Mail)&#x20;
* [ ] Document number of participants and feedback in our [monitoring overview](https://docs.google.com/spreadsheets/d/17EuFJD0flqb1JKbsXo4PgCCs4JMYKiPEwwLIM8mSKhU/edit?usp=sharing)
* [ ] (Forward feedback to external speakers)
