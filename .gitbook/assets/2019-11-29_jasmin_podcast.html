  • import .mp4 files & edit them
  • export to .mp4
  • very good & complex software, so far I only use the basics
  • Link

What I learned so far

  • You can always edit out things
  • You can rerecord your own voice
  • BUT: usually sound circumstances are different once you edit
  • Be careful there is no background noise (small stuff matters! E.g. tapping the table while talking)
